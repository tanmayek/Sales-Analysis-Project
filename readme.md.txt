# Sales Performance Analysis

## Overview

This project analyzes sales data to identify key business insights using SQL and Power BI.

## Tools Used

* PostgreSQL (SQL)
* Power BI
* Excel (CSV dataset)

## Key Insights

* West region generates the highest sales
* Technology category drives the most profit
* Furniture has lower profitability due to higher discounts
* A few products (Canon, Fellowes) contribute significantly to revenue
* Consumer segment generates the highest sales
* Sales show monthly variation with peak periods
